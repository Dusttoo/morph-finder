# Morph Finder

A React Native Expo app that walks crested gecko owners through identifying their gecko's morph via a simple guided question flow.

Built by Dusty Mumphrey / Built By Dusty (builtbydusty.com).

Powered by the same genetics engine as ReptiDex (reptidex.com).

## Stack

- Expo (managed workflow)
- Supabase (auth + database)
- TanStack Query (server state)
- Zustand (UI state)
- React Native Web (Vercel deployment)
- TypeScript
- Jest + Testing Library

## Getting started

```bash
cp .env.example .env
# Fill in your Supabase URL and anon key from supabase.com
npm install
npx expo start
```

Press `i` to open in iOS Simulator or scan the QR code with Expo Go.
